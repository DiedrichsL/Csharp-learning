# Csharp-learning

Ce dépôt regroupe mes premiers exercices en C#, réalisés dans le cadre de mon apprentissage de la programmation.

Chaque exercice est un projet console indépendant visant à travailler la logique, les bases du langage C# et la résolution de problèmes simples.

Ce dépôt a pour objectif de montrer ma motivation, ma progression et ma volonté d’apprendre dans le cadre d’une recherche de stage.
